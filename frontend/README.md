# Frontend Scaffold

This folder contains the initial page scaffold for the CVOS frontend.

Current scope:

1. Landing / Home
2. Resume Intake / Onboarding
3. AI Chat Assistant
4. GitHub Import / Enrichment
5. Resume Review / Edit
6. Quality Score / Evaluation
7. Generation / Processing
8. Final Resume Download

These files are framework-agnostic React component stubs. They are not wired to a router yet.
